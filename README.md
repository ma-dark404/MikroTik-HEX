## 📖 **عن الأداة**
 هي أداة أتمتة لتخمين واختراق بطاقات شبكات المايكروتك (Hotspot) عبر هجوم الـ Brute-Force.

---

## ✨ **الميزات الرئيسية**
- ⚡ **هجوم سريع** متعدد الخيوط (Multi-threading)
- 🧠 **اكتشاف تلقائي** للراوتر على الشبكة
- 💻 **تشغيل مباشر** بدون تثبيت بايثون أو مكتبات خارجية
- 📁 **حفظ واستئناف** الهجمات
- 🌍 **دعم كامل** للغتين العربية والإنجليزية

## 🚀 **التثبيت والتشغيل**

```bash
git clone https://github.com/ma-dark404/MikroTik-HEX.git
cd MikroTik-HEX
chmod +x hex_installer.sh
bash hex_installer.sh
```
